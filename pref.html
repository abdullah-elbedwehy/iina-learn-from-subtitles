How to edit: The text below is sent to the AI. You can rewrite any part of it. Words in {{double braces}} are placeholders — they get replaced with live data when you trigger a lookup.

Available placeholders:

  • {{subtitle}} — the current subtitle line
  • {{title}} — movie or episode title
  • {{filename}} — source file name
  • {{time}} — current playback time
  • {{duration}} — total length
  • {{recent_subtitles}} — last N subtitle lines (set above)
